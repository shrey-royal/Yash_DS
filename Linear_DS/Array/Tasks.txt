12/08 - Arrays - 1D

Basic Exercises:

1. Write a C program to find the sum of all elements in an integer array.
   - Input: [3, 7, 2, 8]
   - Output: Sum = 20

2. Create a program to calculate the average of elements in a floating-point array.
   - Input: [2.5, 4.0, 1.5, 3.0]
   - Output: Average = 2.75

3. Implement a function to find the maximum element in an array of integers.
   - Input: [10, 45, 27, 18, 32]
   - Output: Max = 45

4. Develop a C program to reverse the elements of an array in-place.
   - Input: [5, 8, 12, 3, 9]
   - Output: [9, 3, 12, 8, 5]

5. Write a function to count the occurrences of a specific number in an integer array.
   - Input: Array [4, 2, 6, 2, 8, 2, 9], Number 2
   - Output: Occurrences = 3

6. Create a program that checks if an array is sorted in ascending order.
   - Input: [1, 4, 5, 7, 9]
   - Output: Sorted

7. Implement a function that finds the second smallest element in an integer array.
   - Input: [8, 4, 6, 1, 9, 2]
   - Output: Second Smallest = 2

8. Write a C program to remove all duplicate elements from an integer array.
   - Input: [3, 5, 2, 3, 8, 2, 5, 9]
   - Output: [3, 5, 2, 8, 9]

9. Create a function to rotate the elements of an array to the left by a given number of positions.
   - Input: Array [1, 2, 3, 4, 5], Positions 2
   - Output: [3, 4, 5, 1, 2]

10. Implement a program that searches for a specific element in an array and returns its index, or -1 if not found.
    - Input: Array [6, 8, 12, 5, 9], Element 12
    - Output: Index = 2
